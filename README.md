SQL Database Scripts – Coursework Project

📌 About

This repository contains SQL database definitions and scripts created for coursework and database design practice. It includes the schema and associated SQL commands for building and interacting with a relational database structure, as well as supporting files such as reports and screenshots.

The main artefacts are:

- A .sql file with database creation and manipulation statements

- A PDF report documenting the task and analysis

- A screenshot (likely showing results or ER-diagram)

These components demonstrate your understanding of SQL, database structure, and query logic.

✅ How to Use

To run and test the SQL script locally:

1. Install a SQL database server (e.g. PostgreSQL, MySQL, SQLite or SQL Server).

2. Open your SQL editor or command-line client.

3. Load and execute the SQL file to create the database and tables.

4. Review or run additional queries against the created schema.
